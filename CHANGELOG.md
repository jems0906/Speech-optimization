## [0.1.0] - 2026-03-17

### Added
- Real-time ASR API with REST and WebSocket endpoints.
- Runtime config from YAML and environment variables.
- Optimization utilities: dynamic INT8, `torch.compile`, FP16, Torch-TensorRT integration hooks.
- ONNX export script with optional TensorRT engine build.
- Profiling utilities for latency and PyTorch profiler.
- Docker, Nginx, CI pipeline, linting, tests, and coverage gates.
